# ProcDump

### 미니플러스 덤프
미니플러스 덤프(MiniPlus Dump)는 ProcDump에서만 생성되며, 덤프 용량 중에서 가장 큰 비중을 차지하는 실행 이미지가 제외된 [전체 사용자 모드 덤프](#전체-덤프)와 같다. 디버거 프로그램에 실행 이미지 위치를 알려주면 미니플러스 덤프에서도 어셈블리 코드를 확인할 수 있다. 그러므로 미니플러스 덤프는 전체 덤프를 대체할 수 있다.
